# Bitcointalk Script Chrome Extension

إضافة مخصصة لموقع Bitcointalk توفر مجموعة ميزات منها:
- تغيير الثيم (المظهر)
- إخفاء التوقيع/الأفاتار
- تثبيت المواضيع
- عرض سعر البيتكوين والإيثيريوم مباشرة
- وغيرها من الأدوات المفيدة

## طريقة التثبيت اليدوي
1. حمّل الملفات من [GitHub](https://github.com/yourusername/bitcointalk-script)
2. من "إضافات كروم" فعّل "وضع المطور" ثم "تحميل مجلد غير مضغوط" واختر مجلد الإضافة.
